# VECTOR ARENA - Java通信対戦型シューティングゲーム

## 概要
Java (Swing) を用いて開発した、2D見下ろし型の通信対戦シューティングゲームです。
単なる撃ち合いではなく、ラウンドごとの「ローグライク要素（敗者強化システム）」や、ガード・スキル・マップギミックを駆使した戦略的な戦いが楽しめます。

## 動作環境
* **Java Runtime Environment (JRE) 8 以上** がインストールされていること。
* Windows OS（バッチファイルによる起動を想定）

## 起動方法

コンパイルから実行までを自動化したバッチファイルを用意しています。

### 1. 1台のPCでテストプレイする場合
1.  `ServerStart.bat` をダブルクリックしてサーバーを起動します。
2.  `GameStart.bat` をダブルクリックして、1人目のクライアント（Player1/Host）を起動します。
3.  もう一度 `GameStart.bat` をダブルクリックして、2人目のクライアント（Player2/Guest）を起動します。
4.  タイトル画面でPlayer1がマップを選択し、「スタート」を押すとゲームが始まります。

### 2. 複数のPC（LAN）で対戦する場合
同一のWi-Fiやルーターに接続されている必要があります。

1.  **サーバー役のPC（ホスト）** のIPアドレスを調べます。
    * コマンドプロンプトで `ipconfig` を実行し、「IPv4 アドレス」を確認してください（例: `192.168.x.x`）。
2.  ソースコード内の `src/game/GameConstants.java` をテキストエディタで開きます。
3.  `SERVER_IP` の値を、手順1で調べたIPアドレスに書き換えて保存します。
    ```java
    public static final String SERVER_IP = "192.168.1.15"; // 例
    ```
4.  **サーバー役のPC** で `ServerStart.bat` を起動します。
5.  **それぞれのPC** で `GameStart.bat` を起動します。
    * ※ 起動時にファイアウォールの警告が出た場合は「アクセスを許可」してください。

## 操作方法

キーボードとマウスを使用します。

* **移動**: `W`, `A`, `S`, `D` キー
* **照準**: マウスカーソル
* **射撃**: 左クリック（長押し連射不可、クリック連打が必要）
* **ガード**: 右クリック
    * 前方からのダメージを大幅に軽減します。
    * 使用中は動けず、クールダウン（再使用待ち時間）があります。

## ゲームルールとシステム

### 勝利条件
* 相手のHPを0にするとラウンド勝利です。
* 先に **5ラウンド** 先取したプレイヤーの完全勝利となります。

### ステージ選択
* ゲーム開始時、**ホスト（Player1）** が戦うステージを選択できます。
* 平原、通路、要塞など、地形によって有利な戦術が変わります。

### 敗者強化システム（ローグライク要素）
ラウンドに敗北したプレイヤーは、次のラウンド開始前に「強化カード」を1枚選択できます。
これにより、負けている側ほどキャラクターが強力になり、逆転のチャンスが生まれます。

* **例:**
    * **Shotgun:** 射程は短いが広範囲に多数の弾を発射する。
    * **Vampire:** 相手にダメージを与えると回復する。
    * **The World:** ガード時に時を止め、周囲の弾を消す。
    * **Self Regen:** ガード成功時にHPを回復する。

## トラブルシューティング

**Q. ダブルクリックしても起動しない**
A. Javaがインストールされていないか、環境変数（PATH）が通っていない可能性があります。コマンドプロンプトで `java -version` と入力して確認してください。

**Q. 通信対戦ができない（相手が入ってこない）**
A. サーバー側のPCのセキュリティソフトやWindowsファイアウォールが接続をブロックしている可能性があります。一時的にファイアウォールを無効にするか、Javaの通信を許可してください。

**Q. 文字化けする**
A. ソースコードはUTF-8で保存されています。コンパイル時にエンコーディング指定が必要ですが、付属のバッチファイル (`.bat`) は自動的に `UTF-8` でコンパイルするように設定されています。
