【保存版】対戦サーバー運用チートシート
1. サーバーを起動する（遊ぶとき）
① 自分のPCのターミナル（PowerShell）で実行 まずはAWSのサーバーに入ります。 ※ <IPアドレス> は 13.208.87.43（現在の固定IP）に置き換えてください。

PowerShell

ssh -i "C:\Users\hirot\Desktop\Mykey.pem" ec2-user@13.208.87.43
② サーバーの中（黒い画面）で実行 プログラムをコンパイルし、バックグラウンドで起動します。

Bash

# 念のためコンパイル（更新用）
javac MyServer2.java

# 起動（ログアウトしても止まらないようにする）
nohup java MyServer2 &
これでもう遊べます！PCのターミナルは閉じてしまっても大丈夫です。

2. サーバープログラムを止める（再起動・更新するとき）
① サーバーの中（黒い画面）で実行 動いているJavaの「プロセスID（番号）」を探します。

Bash

ps -ef | grep java
② 番号を確認して停止 表示された結果の、左から2番目あたりの数字（例: 12345）を使います。

Bash

kill <調べた番号>
※ kill 12345 のように入力します。

3. サーバー自体を完全に止める（長期間遊ばないとき）
AWSコンソール（Webブラウザ）で行います。

EC2の画面を開く。

MyServer をチェック。

[インスタンスの状態] → [インスタンスを停止] をクリック。

⚠️ 注意: 「終了」を押すとサーバーが消滅するので絶対に押さないこと！

おまけ：ログ（動作状況）を見たいとき
「誰が接続してきたか」などのログをリアルタイムで見たいときは、サーバー内でこのコマンドを使います。

Bash

tail -f nohup.out
（終了するには Ctrl + C を押します）
